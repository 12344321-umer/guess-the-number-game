# 🎯 Guess My Number Game

This is a fun, retro-style logic game where the player tries to guess a secret number between 1 and 20.

## 🔗 Live Demo
[Click here to play the game!](https://12344321-umer.github.io/guess-the-number-game/)

## 🎮 How to Play
1. Enter a number in the input box.
2. Press the **Guess!** button or hit **Enter**.
3. If your guess is too high or too low, the game will tell you.
4. If you guess correctly, the screen turns **Green**!
5. If you run out of guesses, the screen turns **Red**.
6. Press **Again!** to reset and play a new round.
